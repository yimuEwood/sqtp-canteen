# 2026-05-08 工作日志

## SQTP食堂营养数据网站 - 初始版本搭建完成

- 项目路径：`c:\Users\con83\Desktop\SQTP网站搭建\app\`
- 技术栈：React + TypeScript + Vite + Tailwind CSS + Recharts
- 开发服务器：`http://localhost:5173/`（在 `app/` 目录运行 `npx vite --port 5173 --host`）

### 已实现功能
1. **首页**：英雄区、统计卡片（餐品数/均热量/均价格/均蛋白质）、性价比之王 & 营养评分最高卡片、4张概览图表
2. **餐品数据页**：可搜索/筛选（名称、分类、食堂、价格上限）的数据表格，支持排序、行展开查看详细营养、编辑/删除操作
3. **数据可视化页**：6张图表（分类均热量、类别饼图、性价比 Top8、价格vs热量散点图、宏量对比柱图、营养评分分布）
4. **添加/编辑弹窗**：全字段可编辑，性价比支持自动计算
5. **关于项目页**：项目介绍、研究方法、成员

### 样例数据
已内置12种餐品作为演示数据（3食堂、多分类），可直接在界面增删改。
